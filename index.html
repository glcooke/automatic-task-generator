Генерация задач

Данный генератор позволяет автоматизировать процесс составления задач по информатике для учеников 10-11 классов. Для того, чтобы сгенерировать задания, выберите задачу и сложность на панели слева, введите ключ, выберите количество заданий и нажмите на кнопку "Сгенерировать". Ключом является любое число. По одинаковым ключам всегда будут генерироваться одинаковые задачи. Для генерации случайного ключа нажмите на кнопку "R", для того, чтобы скопировать ключ в буфер обмена, нажмите на кнопку "С". Ответы на задания можно получить и спрятать, нажав на соответствующие кнопки под ними или на боковой панели после генерации.

Печать заданий

В данном генераторе имеется возможность получения файла в формате PDF, содержащем в себе задачи, сгенерированные на данный момент. Для этого сгенерируйте задачи, после чего нажмите на кнопку "Распечатать задания в формате PDF". После этого скачается файл, содержащий в себе задания без ответов.

Генерация контрольных

Для удобства реализована возможность генерировать задачи в формате контрольных. Для этого введите ключ и нажмите на кнопку "Создать контрольную". В появившемся окне введите количество вариантов, и какие задачи будет содержать контрольная, после чего в появившихся окошках введите количство заданий соответстующего типа. В случае, если ничего не вводить, задачи такого типа генерироваться не будут. Поля "Выбор задания" и "Сложность" на генерацию контрольной не влияют. После этого будут сгенерированы варианты с заданиями и спрятанными ответами. Также имеется возможность получить файл с контрольной в формате PDF, нажав на кнопку "Распечатать задания в формате PDF". Отдельные варианты будут на отдельных страницах.
